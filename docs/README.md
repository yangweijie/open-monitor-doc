# open monitor


## 它是什么

一个基于 **ThinkPHP** 的 开源 Web 应用 监控面版。

## 它有什么作用

监控你的应用耗时，以及记录并展示你的异常。


## 灵感

来自于国外的 [inspector](http://app.inspector.dev) 网站。

先是基于其官方php包 做了一个 `think-inspector` 扩展包。

后群友提醒是收费的，于是乎开发了功能 大体 和其一致的面版网站。
